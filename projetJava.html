Un algorithme d'apprentissage automatique

Dans le cadre d'un travail en binôme, nous avions pour objectif de réaliser un programme codé en JAVA visant à trier des dépêches de journaux à l'aide d'autres dépèche déjà triées. Ce projet consistait en différentes étapes :

  • La première consistait en la création d'une base de données où nous associons des mots choisis à des poids, cette étape consistait en un apprentissage supervisé et non automatique
  • La deuxième était un algorithme de tri automatique basé sur une liste de dépèches déjà triées afin de déterminer la catégorie de nouvelles dépèches, l'algorithme associais seul les poids et les mots ensembles.
  • La dernière était une étape d'optimisation afin d'appliquer les méthodes réalisées en cours comme par exemple celle-ci : 
            
            
            
              public static ArrayList<PaireChaineEntier> initDico(ArrayList<Depeche> depeches, String categorie){

                int i = 0;

                String pattern = "[a-z]+";
                while (i < depeches.size()){                                                                                                          //Parcours complet de toutes les dépèches
                    if(depeches.get(i).getCategorie().compareToIgnoreCase(categorie) == 0){                                                           //Si la catégorie est la même que celle de la dépêche
                        j = 0;
                        while (j < depeches.get(i).getMots().size()){
                            boolean matches = Pattern.matches(pattern, depeches.get(i).getMots().get(j));
                            if(matches & UtilitairePaireChaineEntier.indicePourChaine(resultat, depeches.get(i).getMots().get(j)) < 0){               //Si le mot n'est pas déjà dans resultat
                                resultat.add(new PaireChaineEntier(depeches.get(i).getMots().get(j), 0));                                             //Ajoute à l'Arraylist tous les mots
                            }
                            j++;
                        }
                    }
                    i++;
                }
                return resultat;
